# Milestone Review Playbook

Use this at a milestone checkpoint to assess completeness, quality, risk, and
go/no-go confidence.

## Inputs

- milestone definition
- sprint reports
- open bugs
- risk register
- code health indicators

## Process

1. Load the current milestone and related sprint data.
2. Assess feature completeness.
3. Assess bug and test status.
4. Assess risk, velocity trend, and remaining effort.
5. Produce a milestone review and recommendation.

- primary: `agents/directors/producer.md`
