# Skill Test Playbook

Use this to review the Codex playbooks and plugin skill wrappers themselves for
structural and behavioral quality.

## Scope

- markdown playbooks in `playbooks/`
- plugin wrappers in `plugins/openai-codex-game-studios/skills/`

## Modes

- static
- spec
- category
- audit

## Process

1. Parse the mode and scope.
2. Run structural checks on the target playbook or wrapper.
3. Review whether the workflow is complete, collaborative, and has a handoff.
4. Report failures and warnings.

## Recommended Roles

- primary: `agents/leads/qa-lead.md`
- consult: `agents/directors/producer.md`
