# Bug Triage Playbook

Use this to prioritize the open bug backlog and assign bugs to the current
sprint or backlog.

## Modes

- sprint
- full
- trend

## Process

1. Load the bug backlog.
2. Load sprint capacity if available.
3. Reassess severity versus priority.
4. Detect systemic trends and repeated domains.
5. Produce a triage report and assignment recommendation.
