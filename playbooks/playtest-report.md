# Playtest Report Playbook

Use this to create a new playtest report template or to turn raw notes into a
structured report.

## Modes

- new
- analyze

## Process

1. Parse whether this is a blank template or note analysis.
2. Capture session info, focus, first impressions, pain points, delight moments, and bugs.
3. Classify findings by severity and by user understanding.
4. Write the report in `production/playtests/`.

## Recommended Roles

- primary: `agents/leads/qa-lead.md`
- consult: `agents/directors/creative-director.md`
- consult: `agents/directors/producer.md`
