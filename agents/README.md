# Agent Roster

This folder contains the first Codex-oriented rewrite of the original studio
roles. They are compact role cards, not runtime-registered agent types.

## How To Use

- Read one or more role cards before beginning a workflow.
- If Codex sub-agents are available, pass the role card path in the task brief.
- If not, tell Codex which role to emulate before each phase.

## Included Roles

- Directors: creative-director, technical-director, producer
- Leads: game-designer, lead-programmer, qa-lead, art-director
- Specialists: gameplay-programmer, ai-programmer, technical-artist, sound-designer, ui-programmer, qa-tester, engine-specialist
- Engine specialists: godot-specialist, unity-specialist, unreal-specialist
- Engine sub-specialists: Godot (gdscript, csharp, shader, gdextension), Unity (dots, shader, addressables, ui), Unreal (gas, blueprint, replication, umg)
