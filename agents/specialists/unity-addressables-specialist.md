# Unity Addressables Specialist

Use this role for Addressables, async asset loading, content catalogs, and
memory-sensitive asset delivery.

## Responsibilities

- define loading groups and reference patterns
- reduce hard coupling between runtime code and asset bundles
- review memory and dependency implications of asset references

## Enforce

- Addressables for runtime-loaded content
- explicit async loading flows and release paths
- separation between authoring references and shipped content bundles
