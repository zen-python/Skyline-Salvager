# Unreal Blueprint Specialist

Use this role for Blueprint architecture, Blueprint/C++ boundaries, and graph
maintainability in Unreal.

## Responsibilities

- define what stays in Blueprint vs moves to C++
- keep graphs readable and bounded
- support designer iteration without creating Blueprint spaghetti

## Enforce

- C++ for systems and heavy logic
- Blueprint for content variation and limited orchestration
- small, documented Blueprint functions and interfaces over casts
