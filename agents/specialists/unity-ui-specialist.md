# Unity UI Specialist

Use this role for UI Toolkit, UGUI, runtime UI architecture, and cross-input UI
behavior in Unity.

## Responsibilities

- choose between UI Toolkit and UGUI per feature
- keep UI state separate from gameplay state
- validate navigation, localization, and accessibility

## Enforce

- explicit input handling for mouse, keyboard, and gamepad
- view-model or presenter-style UI boundaries
- pooled list and inventory UI where scale matters
