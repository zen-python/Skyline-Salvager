# Game Concept: Skyline Salvager

*Created: 2026-04-13*
*Status: Approved*

## Elevator Pitch

Skyline Salvager is a top-down action-puzzle game where you steer a magnetic
salvage rig across unstable rooftops, pulling valuable cargo into extraction
lanes before the skyline collapses around you.

## Core Identity

| Aspect | Detail |
| --- | --- |
| **Genre** | Action puzzle with light extraction strategy |
| **Platform** | PC |
| **Target Audience** | Players who enjoy Into the Breach, SteamWorld Heist, and tactical score-chasing loops |
| **Player Count** | Single-player |
| **Session Length** | 15-25 minutes |
| **Monetization** | Premium |
| **Estimated Scope** | Small |
| **Comparable Titles** | Into the Breach, Superhot, SteamWorld Dig 2 |

## Core Fantasy

The player feels like an expert urban salvage operator making fast, deliberate
decisions under structural pressure. Every crate pulled to safety feels like a
smart rescue from a machine-sized disaster.

## Unique Hook

The magnetic beam that saves cargo also destabilizes the environment. The
player's main tool is both the solution and the source of risk.

## Core Loop

- **Moment-to-moment**: aim the beam, pull cargo, reposition the rig, and avoid falling debris
- **Short-term**: clear a salvage zone while preserving a safe route to extraction
- **Session-level**: complete 3-5 rooftop chains, bank score multipliers, and unlock higher-risk jobs
- **Long-term**: unlock new beam modules, hazard modifiers, and salvage contracts

## Game Pillars

### Pillar 1: Tense Spatial Problem Solving

Every action should alter the tactical layout in a readable way.

### Pillar 2: Tools With Trade-Offs

The player's best move should often create a new risk that must be managed.

### Pillar 3: Readable Collapse

Hazards must feel threatening without feeling random. The player should usually
understand why a roof section failed.

## Anti-Pillars

- Not a loot treadmill RPG
- Not a precision twin-stick shooter
- Not a fully procedural sandbox with unclear goals
